Northbridge Save Manager SDK.

This tool allows the developers to implement user friendly features (import, export, backup and restore save files) in a seamless and easy way. While the tool was designed for RPG Maker VX Ace games, developers can take the source code and adjust it so it can work for their engine of their choice.

Features

-Set it up and build.

-Supports RPG Maker XP, VX, Ace* and MV*. ( = Requires script. **= Requires a plugin)

-Seamless operation.

-Compatible with Microsoft's Project Centennial/Desktop App Converter.

-Lightweight and unobtrusive. Your players won't notice that it's running in the background.

-Open source. Covered under GPLv3 license.
